# Comfy

岡山市南区泉田のまつげエクステ・増毛エクステサロン「Comfy」のリニューアル用静的サイトです。

WordPressは使わず、通常のHTML/CSS/JavaScriptだけで構成しています。

## Files

- `index.html` - トップページ
- `contact.html` - ご予約・お問い合わせページ
- `eyelash-price.html` - まつ毛エクステ料金ページ
- `volume-price.html` - 増毛エクステンション価格表ページ
- `styles.css` - 共通スタイル
- `script.js` - ヘッダーとスマホナビの動き
- `assets/` - サイト内で使う画像

## Preview

ブラウザで `index.html` を開くと確認できます。
